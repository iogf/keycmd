keycmd

A modal file manager. Keycmd is a very flexible file manager, probably the most flexible one that exists.

Overview

A file system can be abstracted as a tree. Subdirectories can be seen as nodes of such a tree. Keycmd has the feature of viewing a file system as a set of nodes, such nodes correspond to directories of the tree. The nodes are grouped into a sequential style.
